# the-rute-cause

> "Investigating why my workflow actually works."

After a year of using pre-built configurations like HyDE, I realized I was a guest in my own operating system. This repo is the result of me burning it all down and finding the **root** cause of my own productivity.

No bloat. No scripts I don't understand. Just my Arch Linux setup, built from the ground up.
